# The Protocol Settings Events contract. (ProtocolSettingsEvents.sol)

View Source: [contracts/events/ProtocolSettingsEvents.sol](../contracts/events/ProtocolSettingsEvents.sol)

**↗ Extends: [ModulesCommonEvents](ModulesCommonEvents.md)**
**↘ Derived Contracts: [ISovryn](ISovryn.md), [ProtocolSettings](ProtocolSettings.md)**

**ProtocolSettingsEvents**

This contract code comes from bZx. bZx is a protocol for tokenized
margin trading and lending https://bzx.network similar to the dYdX protocol.
 * This contract contains the events for protocol settings operations.
